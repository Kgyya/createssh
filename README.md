# CreateSSH readyssh.net
### Feature:
+ Support SSL (OpenSSH & Dropbear)
+ Support Squid Proxy
+ Support Squid Proxy + SSL
+ Expired 10D - 3D
+ Fullspeed
### Instalation:
```
apt update && apt upgrade
apt install python
apt install git
apt install python-pip
pip install requests
pip install bs4
git clone https://github.com/Kgyya/createssh
cd createssh
python3 main.py
```
